## Overview

Challenges are special tasks created by server admins that reward members with bonus XP and special rewards for completing specific actions or milestones.

## Challenge Types

### Message Challenges

- **Message Count**: Send X messages
- **Daily Goals**: Send messages every day
- **Streaks**: Maintain consecutive days
- **Channel-Specific**: Messages in specific channels

### Reaction Challenges

- **Reaction Count**: Add X reactions
- **Reaction Variety**: Use different types
- **Target Reactions**: React to specific content

### Voice Challenges

- **Voice Time**: Spend time in voice
- **Voice Streak**: Join voice daily
- **Voice Participation**: Active voice engagement

### Social Challenges

- **Invite Friends**: Bring new members
- **Engagement**: Interact with channels
- **Community Events**: Participate in events

### Custom Challenges

Admins can create challenges for:

- Specific milestones
- Special occasions
- Unique server goals
- Event participation

## Completing Challenges

### How It Works

1. **View Challenges**: See available challenges
2. **Start Challenge**: Begin working toward goal
3. **Track Progress**: Monitor your progress
4. **Complete**: Finish the challenge
5. **Receive Rewards**: Get XP and rewards

### Progress Tracking

- **Progress Bar**: Visual progress indicator
- **Current/Goal**: See how much you've done
- **Time Remaining**: Countdown for time-limited challenges
- **Real-Time Updates**: Progress updates automatically

## Challenge Rewards

### XP Rewards

- **Bonus XP**: Extra XP beyond normal activity
- **Variable Amounts**: Different challenges offer different XP
- **One-Time**: Usually awarded once per challenge

### Special Rewards

Some challenges offer:

- **Discord Roles**: Special roles
- **Shop Credits**: Extra XP for shop
- **Badges**: Achievement badges
- **Custom Rewards**: Server-specific rewards

## Challenge Features

### Time-Limited Challenges

- **Start Date**: When challenge begins
- **End Date**: When challenge ends
- **Urgency**: Creates motivation to complete
- **Extensions**: Admins may extend deadlines

### Recurring Challenges

- **Daily Challenges**: Reset every day
- **Weekly Challenges**: Reset every week
- **Monthly Challenges**: Reset every month

### Challenge Series

- **Sequential Challenges**: Challenges that build on each other
- **Challenge Paths**: Series of related challenges
- **Progressive Rewards**: Better rewards as you progress
