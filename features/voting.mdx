## Overview

Voting campaigns let members vote for favorites, and winners automatically receive rewards from a prize pool.

## How It Works

### For Members
1. View active voting campaigns
2. Cast your vote(s)
3. See real-time results
4. Receive rewards if your choice wins

### Campaign Types
- **Single Choice**: Vote for one option
- **Multiple Choice**: Select multiple options
- **Ranked Choice**: Rank options in order

## Prize Distribution

- **Winner Rewards**: Rewards for top vote recipients
- **Participation Rewards**: XP for all voters (sometimes)
- **Automatic**: Rewards given automatically

## Real-Time Results

- See results update live
- View current standings
- See vote counts
- Track voting trends

## Tips

- Vote early to influence results
- Check campaign descriptions
- See what prizes are available
