## Overview

The XP Shop lets you spend accumulated XP on Discord roles, special perks, and custom items.

## Shop Items

### Discord Roles
- Special permissions
- Custom name colors
- Access to exclusive channels

### Perks
- **XP Multipliers**: Earn XP faster
- **Custom Badges**: Display special badges
- **Priority Support**: Faster help

### Custom Rewards
Server-specific items created by admins

## How to Purchase

1. Browse available items
2. Click item to see details
3. Check you have enough XP
4. Click "Buy"
5. Receive item instantly

## Managing Purchases

- View purchase history
- See active items you have
- Check item expiration (if any)

## Tips

- Save XP for valuable items
- Check regularly for new items
- Read descriptions before buying
- Consider XP multiplier items
