## Overview

The XP Shop Management page allows you to create, edit, and manage items that members can purchase with their accumulated XP.

## Creating Shop Items

### Item Types

#### Discord Roles

- **Role Selection**: Choose Discord role to sell
- **Price**: XP cost
- **Description**: What the role provides
- **Availability**: Stock limits (if any)

#### Perks

- **XP Multipliers**: Boost XP earning rate
- **Custom Badges**: Special badges
- **Priority Support**: Faster support
- **Exclusive Access**: Access to special features

#### Custom Rewards

- **Custom Items**: Server-specific rewards
- **Special Privileges**: Unique perks
- **One-Time Bonuses**: Single-use rewards

### Item Configuration

When creating items, set:

- **Name**: Item name
- **Description**: Detailed description
- **Price**: XP cost
- **Category**: Item category
- **Stock**: Available quantity (if limited)
- **Requirements**: Level or role prerequisites
- **Duration**: How long item lasts (if temporary)

## Managing Shop Items

### Active Items

View and manage available items:

- **Edit**: Modify item details
- **Disable**: Temporarily hide from shop
- **Delete**: Remove item permanently
- **Stock Management**: Update stock levels

### Item Analytics

Track item performance:

- **Sales Count**: How many times purchased
- **Total Revenue**: XP earned from sales
- **Popularity**: Most purchased items
- **Sales Trends**: Sales over time

## Shop Settings

### Shop Configuration

Configure shop behavior:

- **Shop Status**: Enable/disable shop
- **Default Categories**: Organize items
- **Purchase Limits**: Limits per member
- **Refund Policy**: Refund rules
