## Overview

Run voting campaigns where members vote for favorites, and winners automatically receive rewards.

## Creating Campaigns

### Setup
- **Campaign Name**: Title of the vote
- **Description**: What members are voting on
- **Options**: Voting choices
- **Prize Pool**: Total rewards to distribute
- **Voting Period**: Start and end dates

### Prize Distribution
- Rewards for winners
- Participation rewards (optional)
- Automatic distribution

## Campaign Types

- **Single Choice**: Vote for one option
- **Multiple Choice**: Select multiple options
- **Ranked Choice**: Rank options in order

## Managing Campaigns

- View active campaigns
- See vote counts and standings
- View results when campaigns end
- Track participation rates

## Tips

- Provide clear voting options
- Set fair prize pools
- Promote campaigns to maximize participation
