## Overview

Voting campaigns allow you to run community votes where members vote for their favorites, and winners receive rewards from a prize pool.

## Creating Voting Campaigns

### Campaign Setup

Configure your voting campaign:

- **Campaign Name**: Title of the voting campaign
- **Description**: What members are voting on
- **Options**: Voting options (e.g., candidates, choices)
- **Prize Pool**: Total XP or rewards to distribute
- **Voting Period**: Start and end dates
- **Vote Limits**: How many votes each member gets

### Prize Distribution

Set up rewards:

- **Winner Rewards**: XP/rewards for winners
- **Participation Rewards**: Rewards for all voters
- **Prize Pool**: Total rewards available
- **Distribution Method**: How prizes are distributed

## Managing Campaigns

### Active Campaigns

View and manage running campaigns:

- **Campaign Status**: Active, upcoming, or ended
- **Vote Counts**: Current vote tallies
- **Leaderboard**: Current standings
- **Time Remaining**: How much time is left

### Campaign Results

View results when campaigns end:

- **Winners**: Top vote recipients
- **Final Results**: Complete vote counts
- **Reward Distribution**: Who received rewards
- **Participation Stats**: How many members voted

## Campaign Types

### Single Choice

Members vote for one option:

- **Best of X**: Choose the best option
- **Election**: Vote for a candidate
- **Preference**: Express preference

### Multiple Choice

Members can vote for multiple options:

- **Top 3**: Select top 3 choices
- **Ranked Choice**: Rank options in order
- **Multiple Selections**: Select multiple favorites

## Campaign Analytics

Track campaign performance:

- **Total Votes**: Number of votes cast
- **Participation Rate**: Percentage of members voting
- **Vote Distribution**: How votes are distributed
- **Trend Analysis**: Voting trends over time
