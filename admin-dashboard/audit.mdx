## Overview

The Security Audit page provides a complete log of all administrative actions, security events, and system changes for accountability and security monitoring.

## Audit Log Features

### Action Logging

All admin actions are logged:

- **User Management**: XP adjustments, role changes
- **Settings Changes**: Configuration modifications
- **Challenge Management**: Challenge creation/editing
- **Shop Management**: Item creation/editing
- **System Changes**: Important system modifications

### Security Events

Track security-related events:

- **Login Attempts**: Successful and failed logins
- **Permission Changes**: Role and permission updates
- **Suspicious Activity**: Unusual patterns
- **Access Logs**: Who accessed what and when

## Viewing Audit Logs

### Filtering Options

Filter logs by:

- **Date Range**: Specific time period
- **Action Type**: Type of action
- **User**: Who performed the action
- **Severity**: Importance level

### Log Details

Each log entry shows:

- **Timestamp**: When action occurred
- **User**: Who performed action
- **Action**: What was done
- **Details**: Specific changes made
- **IP Address**: Where action came from

## Export and Reporting

### Export Logs

- **CSV Export**: Download logs as CSV
- **PDF Reports**: Generate PDF reports
- **Custom Reports**: Create specific reports
