## Overview

Create and manage educational modules to help members learn about your server and community.

## Creating Modules

### Module Types
- **Tutorials**: Step-by-step guides
- **Server Orientation**: Welcome and rules
- **Skill Training**: Teaching specific skills
- **Resources**: Collections of helpful links

### Content
Add to modules:
- Text content
- Images
- Videos
- Links to resources
- Quizzes

## Managing Modules

### Organization
- Group into categories
- Create learning paths
- Set prerequisites
- Recommend order

### Settings
- Visibility controls
- Completion rewards (XP/badges)
- Time estimates
- Difficulty levels

## Analytics

Track module performance:
- Completion rates
- Most popular modules
- Average completion time
- Member progress

## Tips

- Organize modules logically
- Make content engaging
- Offer rewards for completion
- Keep content updated
