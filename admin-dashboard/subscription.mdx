## Overview

View your current plan, manage billing, upgrade or downgrade, and see available features.

## Subscription Plans

### Starter Plan (0-50 members)
- XP & Level tracking
- Public leaderboards
- Basic challenges
- Activity feed
- XP shop system

### Growth Plan (50-250 members)
- Everything in Starter
- Voting campaigns
- Education modules
- Advanced analytics
- Priority support

### Enterprise Plan (250-1000+ members)
- Everything in Growth
- API access
- Custom integrations
- Dedicated support
- Custom features

## Managing Subscription

### Current Plan
- View your plan and billing cycle
- See next billing date
- Check included features

### Upgrading
- Select new plan
- Review feature changes
- Complete payment
- Instant activation

### Downgrading
- Select lower plan
- See effective date
- Review feature changes
- Confirm downgrade

## Billing

- Add or update payment methods
- View billing history
- Download invoices
