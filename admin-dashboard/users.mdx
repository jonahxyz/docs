## Overview

The User Management page provides tools to view all server members, manage their XP, assign roles, and monitor individual member activity.

## Member List

### Viewing Members

Browse all server members with:

- **Search**: Find specific members by username
- **Sorting**: Sort by XP, level, messages, or activity
- **Filtering**: Filter by role, level range, or activity status
- **Pagination**: Navigate through large member lists

### Member Information

For each member, view:

- **Username**: Discord username and avatar
- **Total XP**: Accumulated experience points
- **Level**: Current level
- **Messages**: Total messages sent
- **Rank**: Position on leaderboard
- **Joined Date**: When they joined the server
- **Last Active**: Last activity timestamp

## Member Actions

### XP Management

Manually adjust member XP:

- **Add XP**: Award bonus XP to members
- **Remove XP**: Deduct XP if needed
- **Set XP**: Set XP to specific amount
- **Bulk Actions**: Adjust XP for multiple members

### Role Management

Manage member roles:

- **Assign Roles**: Give Discord roles to members
- **Remove Roles**: Take away roles
- **Level-Based Roles**: Auto-assign based on level
- **Custom Roles**: Assign special roles

### Member Moderation

Moderation actions:

- **View History**: See member's activity history
- **Warnings**: Issue warnings (if configured)
- **Mute**: Temporarily mute member
- **Ban**: Ban member from server

## Member Analytics

### Individual Stats

View detailed stats for each member:

- **Activity Timeline**: Activity over time
- **Message History**: Recent messages
- **XP History**: XP gains over time
- **Challenge Progress**: Challenges they're working on
- **Shop Purchases**: Items they've bought

### Activity Patterns

Analyze member behavior:

- **Peak Activity Times**: When they're most active
- **Channel Preferences**: Favorite channels
- **Engagement Trends**: Activity trends over time

## Bulk Operations

### Bulk XP Adjustments

Adjust XP for multiple members:

- **Select Members**: Choose multiple members
- **Bulk Add XP**: Add XP to all selected
- **Bulk Remove XP**: Remove XP from all selected
- **Bulk Set Level**: Set levels for multiple members

### Bulk Role Management

Manage roles for multiple members:

- **Assign Roles**: Give role to multiple members
- **Remove Roles**: Remove role from multiple members
- **Level-Based Assignment**: Auto-assign based on criteria

## Search and Filters

### Search Functionality

Find members quickly:

- **Username Search**: Search by Discord username
- **User ID Search**: Search by Discord user ID
- **Advanced Filters**: Multiple filter criteria

### Filter Options

Filter members by:

- **Level Range**: Members within level range
- **XP Range**: Members within XP range
- **Role**: Members with specific roles
- **Activity Status**: Active, inactive, or new members
- **Join Date**: Members who joined in date range

## Member Export

Export member data:

- **CSV Export**: Download member list as CSV
- **Selected Members**: Export only selected members
- **Custom Fields**: Choose which data to export
