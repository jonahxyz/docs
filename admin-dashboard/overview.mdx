## Introduction

The Admin Dashboard provides tools to manage your Discord server, configure XP systems, create challenges, manage users, and analyze engagement.

## Access Requirements

You must be:
- Server owner, OR
- Have Administrator permissions
- Have an active subscription (for premium features)

## Key Features

### Server Management
- Multi-server support
- Server selector for switching
- Quick stats overview

### XP Configuration
- Custom XP rates
- Cooldown settings
- Level system customization
- Automatic role rewards

### Challenge System
- Create custom challenges
- Track completion rates
- Set rewards
- View analytics
