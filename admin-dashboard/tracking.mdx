## Overview

Create custom invite links and track which members joined through specific sources.

## Creating Links

### Setup
- **Link Name**: Descriptive name
- **Custom Code**: Unique identifier
- **Destination**: Discord invite link
- **Campaign**: Associate with campaigns

## Link Analytics

Track performance:
- Click count
- Members who joined
- Conversion rate
- Top referrers

## Managing Links

- Edit link details
- Disable or delete links
- Copy shareable links

## Tips

- Create separate links for different campaigns
- Monitor which links perform best
- Consider rewarding top referrers
