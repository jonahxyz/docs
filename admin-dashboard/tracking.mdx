## Overview

Tracking Links allow you to create custom invite links and track which members joined through specific referral sources.

## Creating Tracking Links

### Link Configuration

Set up tracking links:

- **Link Name**: Descriptive name
- **Custom Code**: Unique identifier
- **Destination**: Discord invite link
- **Campaign**: Associate with campaigns

### Link Analytics

Track link performance:

- **Clicks**: How many times clicked
- **Joins**: Members who joined via link
- **Conversion Rate**: Clicks to joins ratio
- **Top Referrers**: Members who referred most

## Managing Links

### Active Links

View and manage links:

- **Edit**: Modify link details
- **Disable**: Temporarily disable
- **Delete**: Remove link
- **Copy Link**: Get shareable link
