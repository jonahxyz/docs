## Overview

Your admin dashboard shows server activity, member engagement, and key metrics at a glance.

## Dashboard Components

### Server Stats
- **Total Members**: Current member count
- **Active Members**: Active in last 24 hours
- **Total XP**: Combined XP of all members
- **Messages Today**: Today's message count
- **Active Challenges**: Number of running challenges

### Quick Actions
- Create Challenge
- Add Shop Item
- Configure Levels
- View Analytics

### Activity Overview
- Message activity charts
- XP distribution
- Engagement trends
- Peak activity times
