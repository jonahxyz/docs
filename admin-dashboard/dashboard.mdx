## Overview

The Admin Dashboard provides a comprehensive overview of your Discord server's activity, member engagement, and key metrics. It's your central hub for managing your community.

## Dashboard Components

### Server Stats

Quick overview cards showing:

- **Total Members**: Current server member count
- **Active Members**: Members active in the last 24 hours
- **Total XP**: Combined XP of all members
- **Messages Today**: Total messages sent today
- **Challenges Active**: Number of active challenges
- **Shop Items**: Available items in the XP shop

### Server Selector

If you manage multiple servers:

- **Switch Servers**: Select which server to manage
- **Server Stats**: View stats for each server independently
- **Quick Access**: Fast switching between servers

### Quick Actions

Common admin tasks accessible from the dashboard:

- **Create Challenge**: Quick access to challenge creation
- **Add Shop Item**: Add new items to the XP shop
- **Configure Levels**: Set up level rewards
- **View Analytics**: Access detailed analytics
- **Manage Users**: User management tools

### Activity Overview

Visual representation of:

- **Activity Chart**: Message activity over time
- **XP Distribution**: How XP is distributed among members
- **Engagement Trends**: Growth and engagement patterns
- **Peak Times**: When your server is most active

### Recent Activity

Feed showing:

- **Recent XP Gains**: Members earning XP
- **Challenge Completions**: Challenges being completed
- **Shop Purchases**: Items being purchased
- **Level Ups**: Members leveling up

## Key Metrics

### Engagement Metrics

- **Daily Active Users**: Members active each day
- **Message Volume**: Total messages per day/week/month
- **XP Distribution**: How XP is spread across members
- **Challenge Participation**: Percentage completing challenges

### Growth Metrics

- **Member Growth**: New members over time
- **Retention Rate**: How many members stay active
- **Engagement Rate**: Percentage of active members
- **XP Growth**: Total XP growth rate

## Navigation

### Admin Sidebar

Access all admin features:

- **Dashboard**: Main overview (current page)
- **Challenges**: Challenge management
- **User Management**: Member management
- **Tracking Links**: Referral link management
- **Analytics**: Detailed analytics
- **Education**: Educational content management
- **Voting**: Voting campaign management
- **Server Support**: Support team management
- **XP Shop**: Shop item management
- **XP Settings**: Configuration
- **Security Audit**: Audit logs
- **Subscription**: Billing and plans

### Admin Search

Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux) to:

- Search for any admin page
- Quick access to features
- Navigate to settings
