## Overview

Configure all aspects of your XP system including rates, multipliers, channel rules, and level rewards.

## Settings Tabs

### XP Events
Create temporary XP multiplier events:
- Double XP weekends
- Special events
- Event duration and multiplier
- Apply to all or specific activity types

### Channel Configuration
Set XP rules per channel:
- Select channels
- Set custom multipliers (0.5x, 2x, etc.)
- Enable or disable XP per channel

### Moderation Settings
- XP penalties for violations
- Mute/ban handling
- Warning system integration

### Voice XP
Configure voice channel tracking:
- Enable voice XP
- XP per minute rate
- Minimum time to earn
- AFK detection

## XP Configuration

### Base Settings
- **XP Range**: Min and max per message (default: 15-25)
- **Cooldown**: Time between XP gains (default: 60 seconds)
- **XP Per Level**: XP needed per level (default: 100)
- **Enable/Disable**: Toggle XP tracking

### Level Rewards
Auto-assign Discord roles at levels:
- Add reward for specific level
- Choose Discord role
- Set level threshold
- Multiple rewards at different levels

## Tips

- Start with default settings
- Use channel multipliers to guide engagement
- Create XP events for special occasions
- Set level rewards early
