## Welcome to NovaXP!

This guide will help you get started with NovaXP, whether you're a server owner managing your community or a member tracking your progress.

## For Server Members

### Signing In

1. Visit your server's NovaXP portal (provided by your server admin)
2. Click **"Sign in with Discord"**
3. Authorize the application
4. You'll be redirected to your dashboard

### Your First Steps

<AccordionGroup>
  <Accordion title="Explore Your Dashboard">
    Your dashboard shows your XP, level, rank, and recent activity. Take a moment to familiarize yourself with the layout.
  </Accordion>
  <Accordion title="Check the Leaderboard">
    See where you rank among all server members. This shows your current position and how much XP you need to move up.
  </Accordion>
  <Accordion title="Browse Challenges">
    Visit the Challenges page to see available challenges you can complete for bonus XP and rewards.
  </Accordion>
  <Accordion title="Visit the XP Shop">
    Check out the shop to see what items you can purchase with your accumulated XP.
  </Accordion>
</AccordionGroup>

## For Server Owners

### Accessing Admin Dashboard

1. Sign in to NovaXP with your Discord account
2. Click **"Admin"** in the sidebar (only visible if you're a server owner/admin)
3. Select your Discord server from the dropdown
4. You'll see the admin dashboard with server stats

### Initial Setup

<AccordionGroup>
  <Accordion title="Configure XP Settings">
    Go to **Admin → XP Settings** to configure:
    - XP range per message
    - Cooldown between XP gains
    - XP required per level
    - Enable or disable XP tracking
  </Accordion>
  <Accordion title="Set Up Level Rewards">
    In **XP Settings → Level Rewards**, assign Discord roles that members automatically receive when they reach specific levels.
  </Accordion>
  <Accordion title="Create Your First Challenge">
    Navigate to **Admin → Challenges** and create a challenge to engage your community. Start with something simple like a daily message goal.
  </Accordion>
  <Accordion title="Add Shop Items">
    Go to **Admin → XP Shop** and add items members can purchase with XP, such as special roles or perks.
  </Accordion>
</AccordionGroup>

## Understanding XP

### How You Earn XP

XP is awarded automatically for:

- **Sending Messages**: 15-25 XP per message (configurable by admin)
- **Reacting**: XP for reacting to messages
- **Voice Activity**: XP for time spent in voice channels
- **Completing Challenges**: Bonus XP from challenges

### Leveling Up

- **Level Calculation**: Your level is based on total XP divided by XP per level
- **Level Rewards**: You automatically receive Discord roles at certain levels (if configured by admin)
- **Notifications**: You'll be notified when you level up

## Navigation Tips

### Using the Sidebar

The sidebar provides quick access to:

- **Dashboard**: Your main overview page
- **Leaderboard**: Server rankings
- **Challenges**: Available challenges
- **XP Shop**: Purchase rewards
- **Education**: Learning modules
- **Settings**: Account preferences
- **Admin**: (Only visible to server owners/admins)

### Keyboard Shortcuts

- **⌘K or Ctrl+K**: Open search to quickly find pages and features
- **Sidebar Navigation**: Click any item to navigate instantly
