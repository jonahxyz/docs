## Overview

The main dashboard is your central hub for tracking XP, viewing stats, and monitoring your progress in the Discord server.

## Dashboard Components

### XP Stats Card

Displays your current XP and level information:

- **Total XP**: Your accumulated experience points
- **Current Level**: Your current level in the server
- **XP to Next Level**: How much XP you need to level up
- **Progress Bar**: Visual representation of your progress to the next level

### Activity Stats

Shows your engagement metrics:

- **Messages Sent**: Total messages in the server
- **Reactions Given**: Total reactions you've added
- **Voice Minutes**: Time spent in voice channels
- **Daily Streak**: Consecutive days of activity

### Recent Activity Feed

Real-time feed showing:

- Recent XP gains
- Challenge completions
- Level ups
- Shop purchases

### Challenges Section

Quick view of:

- **Active Challenges**: Currently available challenges
- **Your Progress**: Challenges you're working on
- **Completed**: Challenges you've finished

### Voting Section

If your server has active voting campaigns:

- **Active Votes**: Current voting campaigns
- **Your Votes**: Votes you've cast
- **Results**: Current standings

### Server Selector

If you're a member of multiple servers:

- Switch between servers
- View stats for each server separately
- Each server has independent XP tracking

## Understanding Your Stats

### XP Calculation

XP is awarded automatically for:

- **Messages**: 15-25 XP per message (configurable by server admin)
- **Reactions**: XP for reacting to messages
- **Voice Activity**: XP for time spent in voice channels
- **Challenge Completion**: Bonus XP from challenges

### Level System

Levels are calculated based on:

- **XP Per Level**: Amount of XP needed per level (default: 100)
- **Exponential Growth**: Higher levels may require more XP
- **Level Rewards**: Automatic role assignments at certain levels

### Activity Tracking

The system tracks:

- **Message Counts**: By hour, day, week, month
- **Activity Patterns**: When you're most active
- **Engagement Trends**: Your participation over time

## Features

### Onboarding Tour

New users see an interactive tour that explains:

- How XP works
- Where to find challenges
- How to use the shop
- Leaderboard navigation

### Onboarding Checklist

A checklist helps you:

- Complete your profile
- Join your first challenge
- Make your first shop purchase
- Explore educational content

### Activity Stats Component

Detailed breakdown of:

- **Last 15 Messages**: Recent message activity
- **Time-based Counts**: Messages in 1h, 6h, 12h, 24h
- **Daily Calendar**: Message count by day
- **Activity Chart**: Visual representation of activity
