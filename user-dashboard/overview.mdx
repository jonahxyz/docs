## Introduction

The User Dashboard is where Discord server members can track their XP, view leaderboards, complete challenges, shop for rewards, and access educational content. This section covers all features available to regular users.

## Accessing the Dashboard

1. Sign in with Discord at your server's NovaXP portal
2. You'll be automatically redirected to your dashboard
3. Use the sidebar to navigate between different sections

## Key Features

### XP Tracking

- **Automatic XP**: Earn XP automatically for messages, reactions, and voice activity
- **Level System**: Level up as you gain XP
- **Activity Stats**: View your message counts, activity streaks, and engagement metrics

### Challenges

- **Active Challenges**: See all available challenges in your server
- **Progress Tracking**: Monitor your progress toward completing challenges
- **Rewards**: Earn XP and special rewards for completing challenges

### XP Shop

- **Shop Items**: Browse available items you can purchase with XP
- **Purchase History**: View your past purchases
- **Instant Delivery**: Receive rewards immediately upon purchase

### Leaderboards

- **Server Rankings**: See top members by XP, level, and activity
- **Your Position**: Track your rank and progress
- **Filtering**: Filter by different time periods and metrics

## Navigation

The sidebar provides quick access to:

- **Dashboard**: Main overview page
- **Leaderboard**: Server rankings
- **Challenges**: Available challenges
- **XP Shop**: Purchase rewards
- **Education**: Learning modules
- **Settings**: Account and preferences
- **Admin**: (Only visible if you're a server admin)
