## Introduction

The User Dashboard is where Discord server members track XP, view leaderboards, complete challenges, shop for rewards, and access educational content.

## Key Features

### XP Tracking
- Automatic XP for messages, reactions, and voice
- Real-time level updates
- Activity stats and streaks

### Challenges
- View available challenges
- Track your progress
- Earn bonus XP and rewards

### XP Shop
- Browse items you can buy
- Purchase with XP instantly
- View purchase history

### Leaderboards
- See server rankings
- Track your position
- Filter by time period

## Navigation

Use the sidebar to access:
- Dashboard
- Leaderboard
- Challenges
- XP Shop
- Education
- Settings
